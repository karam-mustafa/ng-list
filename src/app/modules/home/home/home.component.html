About the task

The goal from this task is obtaining searchable list from Mock Api, that makes you able to see the resources and search on these resources whatever they will be, this task was developed using angular framework (version 14), and I made it in a few hours

Project structure and architecture


The main layer that controls in the application is NgRx Package, I used Store, Select, Effect packages to manage fetching the products and searching on them, I also used Material UI for the Front UI

This project was developed to handle large and distributed projects, by providing modules concepts and makes you able to develop each feature as a module.


Project Module Architecture

Each module will contains:

  • main folder: basically it is angular component it self, and the name of this folder like the module name
  • store folder: this folder contains all state management dependencies, like (reducer, actions, effects, index, and selectors)
  • services folder: this folder contains all required services that this module is depending on
  • tests folder: this folder contains all required unit test
  • models folder: contains all model interfaces for data types
  • module ts file: contains all imports and export for this module
  • route ts file: contains all routes
And you can see that the products module contains all the above folders and files, on the top of that, there is shared module and this module will contains all the shared component, reusable providers, reusable assets and components, imports and exports.
